# AdminisTale — Панель управления серверами Hytale

Десктопное приложение для администрирования серверных инстансов Hytale, построенное на **PySide6** и **PyQt-Fluent-Widgets**.

## Структура проекта

```
administale/
├── main.py                    # Точка входа
├── requirements.txt           # Зависимости
├── core/
│   └── server_manager.py      # Модели данных и бизнес-логика
└── ui/
    ├── main_window.py         # Главное окно (список серверов)
    ├── server_card.py         # Виджет карточки инстанса
    ├── dialogs.py             # Диалоги (авторизация, создание, импорт)
    └── server_dashboard.py    # Панель управления инстансом (вкладки)
```

## Установка

```bash
pip install -r requirements.txt
```

## Запуск

```bash
python main.py
```

## Функциональность

### Главное окно
- Сетка карточек серверных инстансов с онлайном, статусом, аптаймом
- Кнопки создания, импорта и экспорта инстансов
- Автообновление статусов каждые 5 секунд

### Панель управления инстансом

| Вкладка | Описание |
|---------|----------|
| **Главная** | Статус, статистика (CPU/RAM/игроки), быстрые действия (старт/стоп/рестарт), список модов |
| **Консоль** | Потоковые логи в реальном времени, фильтрация по категориям (ошибки/предупреждения/игроки/система), ввод команд с автодополнением |
| **Настройки** | Порт, макс. игроков, название мира, режим игры, JVM-аргументы, автостарт |
| **Файлы** | Древовидный файловый менеджер (аналог проводника) с загрузкой и удалением |
| **Бэкапы** | Создание, восстановление, удаление и скачивание резервных копий |

### Диалоги
- **Авторизация** — вход по логину/паролю с выбором роли (Администратор/Модератор)
- **Создание инстанса** — имя, порт, версия, режим игры
- **Импорт** — загрузка файла-шаблона (.zip / .json)

## Архитектура

- `ServerManager` — синглтон для управления списком инстансов
- `ServerInstance` — dataclass с полным состоянием сервера
- Все UI-компоненты наследуются от Fluent Widgets (тёмная тема)
- Моковые данные для демонстрации интерфейса

## Требования

- Python 3.10+
- PySide6 >= 6.5.0
- PyQt-Fluent-Widgets >= 1.5.0
- Windows 10/11 или Linux (Ubuntu 20.04+)
